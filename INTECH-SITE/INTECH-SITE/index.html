SICAP

SISTEMA DE CONTROLE DE ACESSO PENITENCIARIO

OBJETIVO PRINCIPAL

O sistema tem o propósito de oferecer suporte a equipe que identifica os visitantes de presos nas penitenciarias atraves de reconhecimento facial e assim facilitar o processo de identificar os visitantes.

METAS

Construir uma aplicação java que gerencie o controle de acesso a visitantes via reconhecimento facial

CRONOGRAMA BÁSICO DO PROJETO

O projeto tem um tempo estimado de 12 meses com entrega do relatório final no dia 17/06/2016.
